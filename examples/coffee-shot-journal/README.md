# Example — coffee-shot-journal project rules

Copy these three files to your repo's `.cursor/rules/`:

- `require-tests.mdc` — Vitest, RTL, `npm run test:run`
- `require-docs.mdc` — README, demo-flow, constitution, specs
- `project-context.mdc` — constitution pointer

Do **not** copy SDLC, commit policy, or ask-when-unclear — those are org rules.

See the live files in the coffee-shot-journal repository.
